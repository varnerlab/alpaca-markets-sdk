Types

Return types used throughout the package. Every typed result also exposes a raw::JSON3.Object field so you can reach fields that aren't surfaced directly on the struct.

Trading

Alpaca.Account — Type
Account

Summary of the authenticated trading account. Returned by get_account.

Selected fields:

  • id, account_number, status, currency
  • cash, buying_power, portfolio_value, equity, last_equity
  • pattern_day_trader, trading_blocked, transfers_blocked, account_blocked
  • raw — the original JSON3.Object payload for fields not surfaced explicitly
source
Alpaca.Asset — Type
Asset

A tradable (or historically tradable) instrument on Alpaca. Returned by list_assets and get_asset.

Selected fields:

  • id, symbol, name, class, exchange, status
  • tradable, marginable, shortable, easy_to_borrow, fractionable
  • raw — the original JSON3.Object payload
source
Alpaca.Order — Type
Order

A trading order. Returned by submit_order, get_order, and list_orders.

Selected fields:

  • id, client_order_id, symbol, asset_class
  • side ("buy" or "sell"), type, time_in_force, status
  • qty, filled_qty, limit_price, stop_price, filled_avg_price
  • created_at, submitted_at, filled_at (all DateTime in UTC, may be nothing)
  • raw — the original JSON3.Object payload
source
Alpaca.Position — Type
Position

An open position in the account. Returned by list_positions and get_position.

Selected fields:

  • symbol, asset_id, exchange, asset_class, side ("long" / "short")
  • qty, avg_entry_price, current_price, lastday_price
  • market_value, cost_basis, unrealized_pl, unrealized_plpc, change_today
  • raw — the original JSON3.Object payload

Market data

Alpaca.Bar — Type
Bar

An OHLCV bar for a single symbol at a given timestamp. Returned by get_bars and get_latest_bar.

Fields:

  • symbol — ticker
  • t — bar start time (DateTime, UTC)
  • o, h, l, c — open / high / low / close
  • v — volume
  • n — trade count (may be nothing depending on feed)
  • vw — volume-weighted average price (may be nothing depending on feed)
source
Alpaca.Quote — Type
Quote

A level-1 NBBO quote. Returned by get_quotes and get_latest_quote.

Fields: symbol, t, bid_price, bid_size, ask_price, ask_size, bid_exchange, ask_exchange.

source
Alpaca.Trade — Type
Trade

A last-sale print. Returned by get_trades and get_latest_trade.

Fields: symbol, t, price, size, exchange, id (exchange trade id, may be nothing).

Clock

Alpaca.MarketClock — Type
MarketClock

Server-side market clock snapshot returned by get_clock.

Fields: timestamp, is_open, next_open, next_close (all DateTime UTC except is_open::Bool).

source
Alpaca.CalendarDay — Type
CalendarDay

A single trading session in the market calendar. Returned by get_calendar.

Fields: date, open, close, session_open, session_close. The session_* fields include extended-hours sessions when Alpaca provides them.

source

Options

Alpaca.OptionContract — Type
OptionContract

Metadata for a single listed options contract. Returned by list_option_contracts and get_option_contract.

Selected fields:

  • symbol — OCC-encoded option symbol (e.g. "AAPL260117C00150000"), which is the value you pass to submit_order for single-leg options trades
  • underlying_symbol, root_symbol, underlying_asset_id
  • type — "call" or "put"
  • style — "american" or "european"
  • strike_price, expiration_date, multiplier, size
  • tradable, status
  • open_interest, open_interest_date, close_price, close_price_date
  • raw — original JSON3.Object payload
source
Alpaca.OptionGreeks — Type
OptionGreeks

The standard option greeks returned alongside a snapshot. Any field may be nothing when Alpaca is unable to compute it (e.g. deep ITM / OTM with no quote activity).

Fields: delta, gamma, theta, vega, rho.

source
Alpaca.OptionSnapshot — Type
OptionSnapshot

Snapshot of a single option contract: latest trade/quote, minute bar, implied volatility, and greeks. Returned by get_option_snapshots and get_option_chain_snapshot.

Fields:

  • symbol — OCC-encoded option symbol
  • latest_quote, latest_trade — reuse the shared Quote / Trade types
  • minute_bar, daily_bar, prev_daily_bar — Bar values
  • implied_volatility — as a decimal (e.g. 0.23 = 23%)
  • greeks — an OptionGreeks value, or nothing
  • raw — original JSON3.Object payload
source

Errors

Alpaca.AlpacaError — Type
AlpacaError(status, code, message, body)

Raised when the Alpaca REST API returns a non-2xx response. status is the HTTP status code, code is Alpaca's internal numeric error code (when the body parses as JSON with a code field, otherwise nothing), message is the decoded error message, and body is the raw response body for debugging.
